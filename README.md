# Week 9 Example 3 — Higher or Lower

## Setup and Interaction Instructions

To run the sketch locally, open `index.html` in Google Chrome using Live Server.

Click **Higher ▲** or **Lower ▼** to guess whether the next card will be higher or lower. Reach the target score shown in the HUD to advance to the next level. Equal cards count as wrong, so you must commit to one direction.

### Debug Shortcuts

Press **D** to toggle the Debug Panel.

Keyboard shortcuts:

- **1** – Jump to Level 1
- **2** – Jump to Level 2
- **3** – Jump to Level 3
- **S** – Jump to the Start Screen
- **W** – Jump to the Win Screen
- **O** – Jump to the Game Over Screen

### Opening the Chrome Console

- **Windows:** Press `F12` or `Ctrl + Shift + J`, then click the **Console** tab.
- **Mac:** Press `Cmd + Option + J`
